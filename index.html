// FEATURED PROJECT

Power BI • DAX • Power Query

Stroke Risk Analytics Dashboard

Interactive Power BI dashboard built on a healthcare dataset of 5,110 patients to analyze stroke risk indicators and support decision-making through KPI tracking, filtering and visual risk segmentation.

Problem

The goal of this project was to identify which demographic and medical indicators are most strongly associated with stroke occurrence and to present them in a way that is easy to explore for healthcare-focused decision-making.

What I built

I designed an interactive Power BI dashboard with KPI cards, slicers, age-group analysis, BMI categorization, smoking-status breakdowns and scatter analysis of BMI versus glucose. I also used DAX and Power Query to create measures, risk flags and clean reporting views.

Key insights

The dashboard highlighted that stroke prevalence rises strongly in the 60+ age group and is closely associated with hypertension, heart disease, obesity and elevated glucose. The project turned raw patient-level data into a clear risk-analysis story.
